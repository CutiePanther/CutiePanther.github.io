puppeteer

发表于 2020-02-11


Puppeteer 翻译为“操纵木偶的人”，表达的意思就是通过puppeteer来操控浏览器，实现访问页面，进行截图，下载文件，自动进行键盘输入，提交表单，模拟点击等用户行为

简介

Puppeteer 是谷歌开发的Node库，它提供了一个高级 API 来通过 DevTools 协议控制 Chromium 或 Chrome
Puppeteer 默认以 headless 模式运行，但是可以通过修改配置文件运行“有头”模式。Chrome DevTools的接口很复杂，但puppeteer对其进行了封装

运行

参照官网文档下载依赖
npm i puppeteer可能会报错 nodejs.Error: Chromium is not download
这是因为网络问题不能下载Chromium，设置淘宝镜像可以解决
SET PUPPETEER_DOWNLOAD_HOST=https://storage.googleapis.com.cnpmjs.org
